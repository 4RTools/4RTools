## Running the project
This project was created using Visual Studio 2022, just open `4RTools.sln` in Visual Studio, and you'll be able to run and generate your own releases.

- [x] Autopot
- [x] Autobuff status
- [x] Manage Profiles
- [x] AHK Spammer
